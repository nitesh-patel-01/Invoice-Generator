📄 InvoiceMaker Pro – Free Invoice Generator

A modern, fully responsive, single-file invoice generator that allows users to create, preview, and download professional invoices instantly — without signup or backend.

🌐 Live Demo: https://invoice-generator-neon-alpha.vercel.app/

🚀 Features
✅ Create professional invoices in seconds
✅ Download invoice as PDF
✅ Add GST, HSN/SAC, Tax & Discount
✅ Upload business logo
✅ Multiple currency support (₹, $, €, etc.)
✅ Live preview (real-time updates)
✅ Print-ready layout
✅ Copy invoice as text
✅ Invoice status (Paid / Unpaid / Overdue)
✅ Fully responsive (Mobile + Desktop)
✅ 100% client-side (No data stored)
🧠 Tech Stack
HTML5
CSS3 (Modern UI + Responsive Design)
Vanilla JavaScript
Libraries Used:
html2canvas – Convert HTML to image
jsPDF – Generate PDF
📦 How It Works
Fill invoice details (business, client, items)
Add taxes, discounts, and notes
Preview updates instantly
Click:
Download PDF → Save invoice
Print → Print or save as PDF
Copy → Copy invoice text
